# 03 NFSP: Deep Reinforcement Learning from Self-Play in Imperfect-Information Games

## 这篇解决什么问题

NFSP 处理的是多智能体自我博弈里一个很现实的问题：普通 DQN 在对手也变化的环境中容易追逐非平稳目标，策略会振荡甚至发散。论文把 fictitious play 和深度 RL 结合，用“最佳响应”和“平均策略”两套网络稳定学习。

## 方法拆解

一个 NFSP agent 有两套模型和两类记忆：

- 强化学习网络学习 best response，近似当前对历史平均对手的最优反应。
- 监督学习网络学习 average strategy，拟合自己历史行为的平均。
- RL 记忆一般是近期经验窗口。
- SL 记忆用 reservoir sampling 保存长期行为样本，避免只记住最近策略。

行为策略是混合的：大部分时间按平均策略走，少量时间按最佳响应走。这个少量比例就是 anticipatory parameter。论文实验里较小的 anticipatory 参数更稳定，过大时会破坏 fictitious play 的平滑性。

## 实验和结论

在 Leduc Hold'em 中，NFSP 的 exploitability 下降并接近 Nash 均衡；普通 DQN 的贪心策略高度可利用，平均行为也不稳定。论文给出的解释很重要：DQN 产生的数据分布高度相关且变化剧烈，而 NFSP 的平均策略让经验分布变化更平滑。

## 对我们数字冰壶的启发

我们的冰壶是完全可观测，但仍是两方博弈，裸 DQN 一样会遇到非平稳对手问题。第一个 DQN 版本训练效果平，是符合这篇论文预期的。

可落地做法：

- 不要只和随机对手训练；建立 opponent pool，包括随机、脚本、旧版本模型、搜索模型。
- 维护“平均策略模型”和“最佳响应模型”，或者简化成策略快照池 + 混合采样。
- 评估时不能只看对随机平均分，要看对不同对手家族是否被克制。
- 训练数据要保留长期分布，不能只用最近失败/胜利样本追涨杀跌。

## 局限

原论文针对不完美信息扑克，数字冰壶不是隐藏信息游戏。我们不一定需要完整 NFSP 架构，但“平均策略稳定自我博弈”和“避免只训练对当前对手的最佳响应”非常适用。
